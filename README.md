# Webpack Template

## Setup Instructions

### 1. Install node.js first - https://nodejs.org
Verify if node is installed 
```
node -v
npm -v
```
#### 2. Initialize NPM
```
npm init -y
```
##### 3. Install Webpack and Required Tools
```
npm install -D `
  webpack `
  webpack-cli `
  webpack-dev-server `
  html-webpack-plugin `
  mini-css-extract-plugin `
  css-loader `
  style-loader
```
